    Nexus - Context Driven Development

    Context Driven Development

    OpenNexus installs a complete context-driven workflow into your repo: slash commands, rules, templates, and skill-ready llms.txt generation grounded in project context.

    Quickstart

    cargo install opennexus
    Open Documentation
    • -Specification: Context files define outcomes with E2E-testable next actions.
    • -Reproducibility: Projects can be recreated from context files only.
    • -Knowledge: Code is disposable, but context preserves the real implementation IP.
